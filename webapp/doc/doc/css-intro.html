什么是CSS？

CSS指层叠样式表。如果把HTML比作网页的骨骼，CSS就是衣服，它会告诉你网页上的东西是什么样的。

常用CSS属性

color background width height font box-shadow 



喵呜！

	color: #f00;
	background: #aaa;
	width: 100px;
	height:100px;
	font: 宋体;
	box-shadow: 0 0 10px;
}

html语义化

根据内容的结构化（内容语义化），选择合适的标签（代码语义化）便于开发者阅读和写出更优雅的代码的同时让浏览器的爬虫和机器很好地解析。

在HTML5出来之前，我们用div来表示页面章节，但是这些div都没有实际意义。（即使我们用css样式的id和class形容这块内容的意义）。这些标签只是我们提供给浏览器的指令，只是定义一个网页的某些部分。但现在，那些之前没“意义”的标签因为因为HTML5的出现消失了，这就是我们平时说的“语义”。

尽可能少的使用无语义的标签div和span

nav header footer section article ...
